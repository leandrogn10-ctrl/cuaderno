# El Cuaderno

Phone-first training-log PWA — single-file (`index.html`), offline-first, GitHub Pages.
Built from the app-shell template (C3). State syncs to a private GitHub gist (`cuaderno.json`).

- **Dev loop**: `python3 -m http.server` → http://localhost:8000/
- **Deploy gate**: headless-Chrome harness — all `HARNESS:` lines in `test-harness.html` must PASS.

Exercise demo images and form instructions from https://github.com/yuhonas/free-exercise-db (public domain, Unlicense) — thank you.
